# Лабораторная 2,3 по дисциплине "Технологии программирования"
Небольшой учебный проект на Django, моделирующий простой интернет-магазин канцелярских товаров. В проекте реализована регистрация пользователей, форма покупки, логика предоставления скидки в 10% в случае дня рождения а также имеется набор автоматических тестов для проверки корректной работы форм.

Постановка задачи:

1. Реализовать Django-приложение «магазин», позволяющее:

регистрировать новых пользователей;

предоставлять скидку в связи с днем рождения;

оформлять покупку через форму с указанием адреса;

автоматически валидировать данные.

2. Написать тесты для форм, проверяющие:

корректность регистрации;

обработку неверных e-mail;

логику скидки к дню рождения;

Краткое описание проекта

Основные компоненты:

1. RegistrationForm

Форма регистрации, принимающая:

username

email

password

birthday

При сохранении — создаёт пользователя и профиль, привязанный к нему (у профиля хранится дата рождения).

2. PurchaseForm

Форма оформления покупки. Обязательные поля:

address


3. Профиль пользователя

При регистрации создаётся отдельная модель Profile, содержащая:

связь OneToOne с пользователем;

дату рождения.

4. Автотесты

Тесты проверяют:

валидность корректных данных формы;

отклонение некорректного email;

корректную обработку дня рождения, совпадающего с текущей датой;

требование обязательного поля address в форме покупки.

5. Используемые технологии

Python 3

Django

Django TestCase
